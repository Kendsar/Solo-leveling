# ⚡ Awakened System

Awakened System is a futuristic, gamified productivity dashboard inspired by the Solo Leveling universe. It transforms daily habits, workouts, and personal goals into an immersive "Game System” where users level up through consistency and discipline.

The application blends self-improvement with game mechanics, allowing users to track their daily progress, gain experience (XP), and visualize their evolution over time in a powerful, system-like interface.

---

## 🧠 Concept

Awakened System reimagines productivity as a progression-based experience:

- Complete daily tasks to gain XP
- Build streaks and maintain consistency
- Track weekly performance through visual dashboards
- Level up and evolve your “Hunter Rank”

The goal is to make discipline feel rewarding, engaging, and game-like.

---

## 🧩 Features (MVP)

- 📊 **Global Progress Tracking** (XP system + progression bar)
- 📅 **Weekly Dashboard** with 7 daily workout cards
- 🔵 **Daily Completion Visualization** (doughnut charts)
- ✅ **Task Management System** (checkbox-based habits)
- 🎯 **Goal Tracking Widgets**
- 🌑 **Futuristic UI** inspired by holographic system interfaces
